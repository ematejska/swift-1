# TensorFlowScalar

A scalar data type compatible with TensorFlow.

``` swift
public protocol TensorFlowScalar: _TensorFlowDataTypeCompatible
```

Types that conform to `TensorFlowScalar` can be used as the `Scalar` associated type of
`Tensor`.
